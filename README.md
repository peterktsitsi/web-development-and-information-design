## Module 2: Introduction to Design Systems: CSS

### Project 2A: Your fisrt Design System

For this project you will:
 **Deconstructing a Website Design System (Simple Process)**

1. **Identify Layout Structure**  
   Look at how the page is divided (header, content areas, sidebar, footer).

2. **Break the Page into Boxes**  
   Treat every section as a **box** using the CSS box model.

3. **Identify Repeating Components**  
   Find elements that repeat (buttons, cards, navigation items, forms).

4. **Analyze Spacing**  
   Observe margins and padding that control the distance between elements.
   
6. **Identify Typography Rules**  
   Note heading sizes, paragraph styles, and text hierarchy.

7. **Identify Color System**  
   List primary colors, background colors, and text colors.

8. **Analyze Layout Behavior**  
   Look at how elements align (rows, columns, grids, centered content).

9. **Check Responsive Changes**  
   See how the layout changes on smaller screens.

10. **Convert Findings into Reusable Classes**  
   Turn patterns into reusable CSS rules and components.

But wait there's more...

Now you must Create you first Design System.

## Creating Your First Website Design System (Portfolio)

1. **Define the Page Structure**  
   Decide the main sections of your site (hero, about, projects, contact).

2. **Choose Basic Typography**  
   Select one font for headings and one for body text.

3. **Define a Simple Color Palette**  
   Pick 3–4 colors (primary, background, text, accent).

4. **Create a Spacing System**  
   Choose consistent spacing values (for example: 8px, 16px, 24px, 32px).

5. **Design Core Components**  
   Create reusable elements like buttons, cards, and navigation links.

6. **Apply the Box Model**  
   Use padding, margins, and borders to control spacing and layout.

7. **Create Layout Rules**  
   Decide how content aligns (centered container, grid for projects, stacked sections).

8. **Make It Responsive**  
   Ensure sections stack neatly on smaller screens.

9. **Reuse the System Across Pages**  
   Apply the same styles to every page to maintain visual consistency.

At this point you have some kind of infinite design canvas, like figma and your design system has been complete.
Time to move on to the next stage, Apply this design system to your website.
